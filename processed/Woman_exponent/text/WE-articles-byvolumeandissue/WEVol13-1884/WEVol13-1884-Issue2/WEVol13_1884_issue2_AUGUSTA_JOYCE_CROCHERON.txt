			AUGUSTA JOYCE CROCHERON.


Some readers are like the hour-glass (their
reading is as the sand; it runs in and runs out
but leaves not a vestige behind); some like a
sponge, which imbibes everything, and returns
it in the same state only a little dirtier; some
like a jelly-bag which allows all that is good
to pass away, and retains only the refuse and
dregs. The fourth class may be compared to
the slave of Golconda, who casting away all
that is worthless,preserves only the pure gems.
—Coleridge.
